Movies CRUD web application developed as part of the **Continuous Integration and Delivery** course.

## Architecture
The application consists of three independent services:
- **Frontend:** React (Vite)
- **Backend:** FastAPI
- **Database:** PostgreSQL

Frontend communicates with the backend via REST API, while the backend persists data in PostgreSQL.

## Tech Stack
- React (Vite)
- FastAPI
- PostgreSQL
- Docker & Docker Compose
- GitHub Actions (CI/CD)
- Kubernetes

## Features
- Create, read, update and delete movies
- RESTful API
- Containerized services
- CI/CD pipeline with Docker image publishing
- Kubernetes deployment
